---
title: "Update Sdk Version"
---

Download the latest SDK from our website and follow these steps:

1. **Standard Update Process**
   - Replace `brd_api.js` with the newer version.
   - Delete the existing service directory and replace it with the new version.
   
{(user.content?.platform === "webos" || user.content?.platform === "tizen") && (
  <>
    - **WebOs:** [see](https://docs.bright-sdk.com/integration-guides/webos-integration#how-to-update-sdk).
    - **Tizen:** [see](https://docs.bright-sdk.com/integration-guides/tizen-integration#how-to-update-sdk).
  </>
)}

2. **Hosted Web App Considerations**
   Important: if your app is [<u>Hosted Web App</u>](https://webostv.developer.lge.com/develop/getting-started/web-app-types) maintain SDK file version consistency across all components.
   Since `brd_api.js` is typically hosted on your web server, ensure backward compatibility with previous app versions before submitting to LG.
   Example:
   - <u>Current version:</u> [www.mywebsite.com/myapp](http://www.mywebsite.com/myapp) (keep old `brd_api.js` here).
   - <u>Updated version:</u> [www.mywebsite.com/myapp/v2](http://www.mywebsite.com/myapp/v2) (use new `brd_api.js`).
   This approach allows older app installations to continue functioning while new versions use the updated SDK.
